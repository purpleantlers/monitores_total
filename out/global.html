products

Array de productos. Cada elemento del array representa un producto y contiene las siguientes propiedades: - `image`: El nombre de la imagen del producto. - `name`: El nombre del producto. - `inches`: El tamaño del producto en pulgadas. - `brand`: La marca del producto. - `price`: El precio del producto. - `quantity`: La cantidad del producto en stock.

Muestra la modal con contenido diferente dependiendo del valor de `modalValue`.

Name Type Description
modalValue string El tipo de modal que se mostrará. Posibles valores: - 'add': Formulario para agregar un producto. - 'update': Formulario para editar un producto existente. - 'delete': Diálogo de confirmación para eliminar un producto existente.
index number (Opcional) El índice del producto a editar o eliminar, utilizado cuando `modalValue` es 'update' o 'delete'.
